1. CNN (Convolutional Neural Network)

CNN이 필요한 이유는 일반적인 신경망으로 이미지를 처리하면 문제가 크다.

CNN은 이를 해결하기 위해 지역 특징, 공간 구조를 유지하면서 학습을 한다.

CNN 핵심 아이디어

CNN은 작은 필터가 이미지를 흝으면서 특징을 찾는다.

“이미지를 작은 영역 단위로 분석하자”

CNN 구조

입력 이미지
   ↓
Convolution Layer (합성곱층)
   ↓
Activation(ReLU) (활성화 함수)
   ↓
Pooling Layer (풀링층) 
   ↓
(반복)
   ↓
Flatten (평탄화)
   ↓
Fully Connected Layer (완전연결층)
   ↓
Output

Convolution Layer (합성곱층)

CNN의 핵심

작은 필터를 사용한다.

Feature Map

Convolution 결과로 생성되는 특징 지도입니다.

입력 이미지
   ↓
필터 적용
   ↓
특징맵(feature map)

필터 하나당 feature map 하나 생성.

필터 수 = 추출 특징 수

Stride

핕터가 이동한 칸 수

큰 stride:

  • 계산 감소
  • 정보 손실 증가

Activation Function

주로 ReLU 사용

왜 필요한가?

  • 비선형성 추가
  • 깊은 네트워크 가능

Pooling Layer

크기를 줄이는 역할.

효과 :

  • 연산량 감소
  • 과적합 감소
  • 주요한 특징 유지

Fully Connected Layer

최종 분류 수행

CNN 장단점

장점

  • 이미지 처리 매우 강력
  • 특징 자동 추출
  • 위치 정보 활용

단점

  • 계산량 큼
  • 데이터 많이 필요
  • 순서 정보 처리 약함

RNN (Recurrent Neural Network)

RNN이 필요한 이유

CNN은 순서를 잘 못 다룬다.

RNN은 이전 상태를 기억한다.

RNN 구조

시간축으로 펼치면

x1 → h1 → y1
     ↓
x2 → h2 → y2
     ↓
x3 → h3 → y3
  • x = 입력
  • h = hidden state
  • y = 출력

Hidden State

핵심 기억 공간

RNN 특징

장점

  • 순서 데이터 처리 가능
  • 문맥 이해 가능

단점

  • 긴 문장 기억 어려움
  • Gradient Vanishing 발생

Gradient Vanishing 문제

역전파 중 gradient가 매우 작아짐

결과

  • 오래된 정보 학습 실패
  • 앞 단어 기억 못함

LSTM (Long Short-Term Memory)

LSTM은 RNN 개선판이다

“중요한 정보를 오래 기억하자”

LSTM 구조

LSTM은 memory cell을 가진다

구성

  • cell state
  • forget gate
  • input gate
  • output gate

Cell State

장기 기억 저장소

정보가 흐르는 통로

Forget Gate

무엇을 잊을지 결정

출력

  • 0 → 버림
  • 1 → 유지

Input Gate

무엇을 저장할지 결정

Output Gate

출력 결정

LSTM 장단점

장점

  • 긴 문맥 기억 가능
  • 번역 성능 우수
  • 시계열 강력

단점

  • 계산량 큼
  • 구조 복잡
  • 느림
